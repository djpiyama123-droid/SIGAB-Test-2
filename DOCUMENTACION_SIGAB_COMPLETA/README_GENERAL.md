# 📚 Centro de Documentación SIGAB

Bienvenido a la carpeta oficial de documentación del proyecto **SIGAB (Sistema Integral de Gestión de Activos Biomédicos)** desarrollado para el **HGR No. 1 IMSS Tijuana**.

Este repositorio contiene toda la información generada durante las sesiones de desarrollo con **Antigravity AI**.

## 📑 Contenido de esta Carpeta

1.  **[MANUAL_TECNICO.md](./MANUAL_TECNICO.md)**: Guía detallada de instalación, dependencias y arquitectura del sistema (FastAPI + React + MySQL).
2.  **[MANUAL_USUARIO_IA.md](./MANUAL_USUARIO_IA.md)**: Instrucciones para interactuar con el Bot de WhatsApp y el Copilot de IA Local.
3.  **[DIAGRAMAS_LOGICOS.md](./DIAGRAMAS_LOGICOS.md)**: Colección de diagramas Mermaid sobre el flujo operativo y la arquitectura de red.
4.  **[PLAN_DE_TRABAJO_MAESTRO.md](./PLAN_DE_TRABAJO_MAESTRO.md)**: Hoja de ruta del proyecto, incluyendo las fases completadas (Tecnovigilancia, KPIs, Almacén, Copilot).

## 🏢 Sobre el Proyecto
SIGAB es una solución **100% On-Premise**, diseñada para ejecutarse en servidores locales sin dependencia de internet, cumpliendo con los estándares de seguridad hospitalaria y la normatividad mexicana (NOM-016, NOM-240).
