## Hallazgos importantes durante ejecución

1. **El `.env` de VPS sobreescribe defaults del `docker-compose.yml`**. Cambiar default sin tocar `.env` del servidor no tiene efecto. Lección: actualizar ambos.
2. **Prefix de rutas es `/api/copilot/`** (no `/api/v1/copilot/`). El plan asumía v1 incorrectamente.
3. **`aiomysql` no permite paralelización con `gather` en una connection** — habría que usar pool. Saltado con justificación.
4. **`num_ctx` en payload override el modelfile** (qwen-claw tiene 16k, request usa 8k → menos RAM, menos prefill time).
5. **Cache funciona correctamente** — turno 2 da TTFT 0.38s (vs 7.2s del turno 1), validando el hit del prefix-cache de Ollama + cache TTL del backend.
